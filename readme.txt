底盘电机：张大头步进
版本：x42_v1.3

底盘安装方式：
	A(4)		B(1)
	B(3)		A(2)
左侧电机和右侧电机的物理安装或接线是相反的。
对于右侧（M1, M2）：给正数指令，轮子向前转。
对于左侧（M3, M4）：给负数指令，轮子才向前转。

想象一个左手坐标系：
当底盘向前运动，由受力分析知，四个轮子都要向前转动，故指令正负依次为++--
当底盘向右运动，由受力分析知，A轮要向后转动，B轮要向前运动，抵消前后方向的力，指令正负依次为+--+
而当底盘逆时针旋转，需要左侧轮子向后，右侧轮子向前，指令为++++
由此得到速度结算公式。
